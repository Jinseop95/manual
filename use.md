### 로컬저장소에서 원격저장소로 올리기
- git init : 깃 설정  
- git status : 바뀐 내용 확인  
- git add . : 전체 추가  
- git commit -m "message" : 커밋 메세지 추가  
- git push origin master : 원격저장소로 올리기  

### 로컬저장소, 원격저장소 파일 삭제  
- git rm 파일명 Or git rm -r 폴더명 : 로컬저장소에서 삭제  
- git commit -m "message" : 커밋 메세지 추가  
- git push origin master : 원격저장소 연동
